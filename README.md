# Ecomora Admin Dashboard

Ecomora is a high-end administrative dashboard interface designed for e-commerce platforms. This project focuses on complex UI layouts, data visualization, and seamless user navigation.

---

## Live Deployment
[Visit the Live Dashboard](https://kanidehbokri.github.io/ecomora-admin-dashboard/)

---

## Technical Features
* **Advanced Data Layouts:** Structured sections for sales analytics, order tracking, and customer management.
* **Component-Based UI:** Custom-designed cards, tables, and sidebars for a cohesive admin experience.
* **Modern CSS Implementation:** Extensive use of CSS Grid and Flexbox for a robust, responsive architecture.
* **Navigation Logic:** Interactive sidebar and header for intuitive user workflow.
